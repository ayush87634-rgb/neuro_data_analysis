# Neuro Data Analysis

A computational neuroscience portfolio built during my research internship 
at IISER Tirupati (Dr. Devanathan's Lab, 2026).

## What This Repository Contains

Jupyter notebooks applying Python-based data analysis to neuroscience 
and neurodegeneration research data.

## Notebooks

### 1. Gene Expression Analysis
Simulated qRT-PCR data analysis across three conditions (Control, 
RA-Differentiated, Treated) for key neurological genes (CHAT, TH, BDNF).

**Skills demonstrated:**
- Data manipulation with pandas and numpy
- Statistical testing (t-test, p-values)
- Publication-quality visualization with matplotlib and seaborn
- Significance annotation (*, **, ***)

### 2. Real Neurodegeneration Dataset (Coming Soon)
Analysis of published GEO dataset from Alzheimer's/Parkinson's research.

### 3. Neurite Morphology Analysis (Coming Soon)
Image-based quantification of neuronal morphology using scikit-image.
